# 🧹 GlassWiper — 体感擦玻璃小游戏 需求文档

## 一、项目概述

**项目名称**：GlassWiper  
**类型**：Web 端体感交互小游戏  
**核心玩法**：通过摄像头实时检测玩家手掌位置，在屏幕上模拟擦玻璃动作，擦干净玻璃上的污渍即可得分。  
**技术栈**：HTML5 + CSS3 + JavaScript + MediaPipe Hands（手部检测）+ Canvas  
**视觉风格**：卡通窗户效果，每个关卡都是不同的窗内/窗外场景

---

## 二、核心功能

### 2.1 摄像头手掌检测
- 调用用户摄像头，使用 **MediaPipe Hands** 进行实时手部关键点检测
- 识别手掌中心位置，映射到游戏画布坐标
- 支持**单手 / 双手**操作（双手擦得更快）
- 检测手掌**张开状态**：只有手掌张开（五指展开）时才视为"擦拭"动作，握拳时不擦拭

### 2.2 窗户场景系统
- 游戏画面设计为**卡通风格的窗户**，带有窗框和窗台装饰
- 每个关卡都是不同的窗内/窗外场景：
  - 室内场景：美女、宠物、家居装饰等
  - 室外场景：城市风景、自然风光、汽车等
  - 特殊场景：节日主题、奇幻场景等
- 污渍层使用 Canvas 绘制，由半透明的灰色/雾气纹理覆盖
- 手掌经过的区域，污渍被擦除，露出下方场景
- 擦除效果带有**自然的笔触感**，边缘柔和，模拟真实擦玻璃的效果

### 2.3 场景元素解锁系统
- 初始关卡只擦除局部关键脏污区域
- 随着擦除进度，逐步解锁场景中的更多元素
- 特殊元素需要达到特定擦除百分比才能完全显现
- 完成关卡后，所有场景元素完全展示

### 2.4 计分系统
- **基础分**：每关擦干净获得 100 分
- **时间奖励**：在限定时间内完成，剩余时间越多奖励越高
- **连击奖励**：连续快速擦拭（手掌持续移动）时触发连击，分数倍率提升（x1.5 / x2 / x3）
- **完美奖励**：如果擦除面积达到 95% 以上，额外奖励 50 分
- **元素发现奖励**：每解锁一个新场景元素获得额外分数

## 三、游戏流程

### 3.1 游戏状态流转

```
[开始界面] → [摄像头授权] → [倒计时 3-2-1] → [游戏进行中] → [场景元素逐步解锁] → [关卡完成] → [下一关 / 游戏结束]
                                                    ↓
                                              [时间耗尽] → [结算界面]
                                                    ↓
                                              [BOSS关卡] → [BOSS战] → [结算界面]
```

### 3.2 开始界面
- 游戏标题 "GlassWiper 🪟"（使用窗户emoji）
- "开始游戏" 按钮
- 窗户风格的界面设计，带有窗帘动画效果
- 历史最高分显示（localStorage 存储）
- 场景画廊预览（已解锁的场景缩略图）

### 3.3 游戏进行中
- 整体设计为**卡通窗户界面**，带有窗框装饰
- 顶部显示：当前关卡、分数、剩余时间、擦除进度条（设计为温度计样式）
- 中间区域：窗户场景画面，污渍覆盖
- 右下角：摄像头小窗口（设计为窗户上的小镜子）
- 手掌位置用**卡通抹布图标**标识
- 场景元素解锁时有**闪光动画**提示

### 3.4 结算界面
- 窗户风格的结算界面
- 显示总分、擦除关卡数、发现的场景元素数量
- 展示当前关卡的完整场景图片
- "再来一局" 按钮

## 四、关卡设计（梯度区分）

### 4.1 关卡难度递进

| 关卡 | 时间限制 | 污渍特点 | 场景类型 | 擦除要求 | 特殊机制 |
|------|---------|---------|---------|---------|---------|
| 第1关 | 90秒 | 局部薄雾（一次擦净） | 室内美女 | 30%关键区域 | 只擦脸部区域解锁美女容貌 |
| 第2关 | 80秒 | 中等污渍（一次擦净） | 窗外跑车 | 50%区域 | 擦出车标和车轮获得奖励 |
| 第3关 | 70秒 | 局部厚污渍（需擦2次） | 宠物房间 | 65%区域 | 只有局部不规则污点，无全屏底色 |
| 第4关 | 60秒 | 局部油渍（需擦3次） | 城市夜景 | 75%区域 | 只有局部油渍斑块，需多次擦拭 |
| 第5关 | 50秒 | 混合污渍（需擦2~3次） | 海滩风景 | 85%区域 | 雾气+油渍混合，多层叠加 |
| 第6关 | 45秒 | 全屏混合（需擦3次） | 奇幻森林 | 90%区域 | 最厚污渍，寻找隐藏魔法生物 |
| BOSS关 | 60秒 | 特殊图案 | BOSS窗外 | 找到BOSS | BOSS藏在污渍中，找到后才能战斗 |

### 4.2 场景元素解锁机制
- **第1关**：逐步解锁美女的面部特征（眼睛→鼻子→嘴巴→完整面容）
- **第2关**：逐步展现跑车的细节（车标→车轮→车身线条→完整跑车）
- **第3关**：发现房间内的隐藏宠物（猫咪在沙发后→狗狗在床下→小鸟在笼中）
- **第4关**：点亮城市的灯光（路灯→霓虹灯→窗户灯光→完整夜景）
- **第5关**：清理海滩的不同区域（沙子→海水→沙滩玩具→夕阳）
- **第6关**：发现森林中的魔法元素（蘑菇→萤火虫→独角兽→魔法树）
- **BOSS关**：BOSS初始隐藏在污渍中，需要找到并擦出才能开始战斗

### 4.3 BOSS关卡设计
- BOSS出现在窗外场景中
- 污渍图案特殊，暗示BOSS的位置
- 玩家需要先找到并擦出BOSS
- BOSS被发现后，进入战斗模式
- 战斗方式：用手掌"拍打"窗外的BOSS
- BOSS有血条，被击中会掉血
- 击败BOSS获得大量分数奖励

## 五、创意玩法 ✨

### 5.1 🎵 音乐节奏擦玻璃（节奏模式）
- 背景播放节奏音乐，污渍会**按节拍出现**在特定位置
- 玩家需要**跟着节奏**擦拭对应位置的污渍
- 踩中节拍擦拭得分翻倍，miss 则扣分
- 类似"太鼓达人"但用手掌擦拭代替敲击

### 5.2 🆚 双人对战模式
- 屏幕分为左右两扇窗户，各有一个场景
- 两个玩家各用一只手（左手控制左边，右手控制右边）
- 谁先擦干净谁赢
- 可以触发"干扰道具"给对方窗户泼脏水 💦

### 5.3 🎨 场景收集系统
- 每关的完整场景会被收录到"场景相册"中
- 集齐特定系列（如：美女系列、跑车系列、风景系列）可解锁特殊抹布皮肤
- 激励玩家反复游玩不同关卡

### 5.4 🌧️ 窗户天气事件
- 游戏过程中随机触发窗户天气事件：
  - **下雨** 🌧️：窗户上不断出现新的雨滴，增加擦拭难度
  - **起雾** 🌫️：已擦干净的区域会慢慢重新起雾
  - **阳光** ☀️：触发后 5 秒内擦拭效率翻倍
  - **小鸟停留** 🐦：小鸟停在窗台上，清理该区域获得奖励

### 5.5 🏪 道具商店
- 游戏中随机掉落或用积分购买道具：
  - **超级抹布**：擦拭范围扩大 2 倍，持续 10 秒
  - **清洁剂喷雾**：点击使用，自动清除一个区域的污渍
  - **时间冻结**：暂停倒计时 5 秒
  - **放大镜**：显示隐藏的场景元素位置

### 5.6 🏆 成就系统
- **美容大师**：完美解锁美女关卡的所有特征
- **汽车专家**：发现跑车关卡的所有细节
- **动物朋友**：找到所有隐藏的宠物
- **夜景摄影师**：点亮所有城市灯光
- **BOSS猎人**：快速找到并击败BOSS
- **场景收藏家**：收集所有场景图片

## 六、UI/UX 设计

### 6.1 视觉风格
- 整体风格：**卡通窗户效果**，带有温馨的居家感
- 主界面：木质窗框 + 窗帘动画 + 窗台装饰
- 色彩方案：暖色调为主，营造舒适的氛围
- 污渍纹理：使用柔和的半透明效果，不影响场景美观
- 污渍形状：采用**不规则形状**（贝塞尔曲线随机变形），避免正圆形的不自然感
- 多层污渍：厚污渍和油渍采用**多层叠加**机制，需要擦多次才能完全清除，每擦一次减少一层
- 擦除动画：带有水痕和闪光效果
- 元素解锁：闪光动画 + 音效提示

### 6.2 音效设计
- 擦拭音效：轻柔的摩擦声
- 元素解锁：清脆的"叮铃"声
- 关卡完成：欢快的音乐片段
- BOSS出现：紧张的音效
- BOSS被击中：打击音效
- 背景音乐：轻松愉快的室内音乐

### 6.3 响应式设计
- 窗户比例固定为 16:9
- 窗框设计适应不同屏幕尺寸
- 支持桌面端浏览器（Chrome / Firefox / Safari / Edge）
- 移动端适配考虑中
